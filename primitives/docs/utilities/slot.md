		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Slot  Merges its props onto its immediate child.  ## Features  - Can be used to support your own \`asChild\` prop.  ## [Installation](#installation)  Install the component from your command line.  ```bash npm install @radix-ui/react-slot ```  ## [Anatomy](#anatomy)  Import the component.  ```jsx import { Slot } from "radix-ui";  export default () =&gt; (   &lt;Slot.Root&gt;     &lt;div&gt;Hello&lt;/div&gt;   &lt;/Slot.Root&gt; ); ```  ## [Basic example](#basic-example)  Use to create your own `asChild` API.  When your component has a single children element:  ```jsx // your-button.jsx  import * as React from "react";  import { Slot } from "radix-ui";  function Button({ asChild, ...props }) {   const Comp = asChild ? Slot.Root : "button";    return &lt;Comp {...props} /&gt;; } ```  Use `Slottable` when your component has multiple children to pass the props to the correct element:  ```jsx // your-button.jsx  import * as React from "react";  import { Slot } from "radix-ui";  function Button({ asChild, children, leftElement, rightElement, ...props }) {   const Comp = asChild ? Slot.Root : "button";    return (     &lt;Comp {...props}&gt;       {leftElement}        &lt;Slot.Slottable&gt;{children}&lt;/Slot.Slottable&gt;        {rightElement}     &lt;/Comp&gt;   ); } ```  ### [Usage](#usage)  ```jsx import { Button } from "./your-button";  export default () =&gt; (   &lt;Button asChild&gt;     &lt;a href="/contact"&gt;Contact&lt;/a&gt;   &lt;/Button&gt; ); ```  ### [Event handlers](#event-handlers)  Any prop that starts with `on` (e.g., `onClick`) is considered an event handler.  When merging event handlers, `Slot` will create a new function where the child handler takes precedence over the slot handler.  If one of the event handlers relies on `event.defaultPrevented` make sure that the order is correct.  ```jsx import { Slot } from "radix-ui";  export default () =&gt; (   &lt;Slot.Root     onClick={(event) =&gt; {       if (!event.defaultPrevented) console.log("Not logged because default is prevented.");     }}   &gt;     &lt;button onClick={(event) =&gt; event.preventDefault()} /&gt;   &lt;/Slot.Root&gt; ); ```
