		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Dropdown Menu  Displays a menu to the user—such as a set of actions or functions—triggered by a button.  ## Features  - Can be controlled or uncontrolled.  - Supports submenus with configurable reading direction.  - Supports items, labels, groups of items.  - Supports checkable items (single or multiple) with optional indeterminate state.  - Supports modal and non-modal modes.  - Customize side, alignment, offsets, collision handling.  - Optionally render a pointing arrow.  - Focus is fully managed.  - Full keyboard navigation.  - Typeahead support.  - Dismissing and layering behavior is highly customizable.  ## [Installation](#installation)  Install the component from your command line.  ```bash npm install @radix-ui/react-dropdown-menu ```  ## [Anatomy](#anatomy)  Import all parts and piece them together.  ```jsx import { DropdownMenu } from "radix-ui";
