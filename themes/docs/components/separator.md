		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Separator  Visually or semantically separates content.  ```jsx &lt;Text size="2"&gt;   Tools for building high-quality, accessible UI.   &lt;Separator my="3" size="4" /&gt;   &lt;Flex gap="3" align="center"&gt;     Themes     &lt;Separator orientation="vertical" /&gt;     Primitives     &lt;Separator orientation="vertical" /&gt;     Icons     &lt;Separator orientation="vertical" /&gt;     Colors   &lt;/Flex&gt; &lt;/Text&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Separator primitive](/primitives/docs/components/separator) and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop          | Type                                     | Default        | | ------------- | ---------------------------------------- | -------------- | | `orientation` | `Responsive&lt;"horizontal" \| "vertical"&gt;` | `"horizontal"` | | `size`        | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;`   | `"1"`          | | `color`       | `enum`                                   | `"gray"`       | | `decorative`  | `boolean`                                | `true`         |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the separator. The largest step takes full width or height of the container.  ```jsx &lt;Flex direction="column" gap="4"&gt;   &lt;Separator orientation="horizontal" size="1" /&gt;    &lt;Separator orientation="horizontal" size="2" /&gt;    &lt;Separator orientation="horizontal" size="3" /&gt;    &lt;Separator orientation="horizontal" size="4" /&gt; &lt;/Flex&gt; ```  ```jsx &lt;Flex align="center" gap="4" height="96px"&gt;   &lt;Separator orientation="vertical" size="1" /&gt;    &lt;Separator orientation="vertical" size="2" /&gt;    &lt;Separator orientation="vertical" size="3" /&gt;    &lt;Separator orientation="vertical" size="4" /&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Separator color="indigo" size="4" /&gt;    &lt;Separator color="cyan" size="4" /&gt;    &lt;Separator color="orange" size="4" /&gt;    &lt;Separator color="crimson" size="4" /&gt; &lt;/Flex&gt; ```  ### [Orientation](#orientation)  Use the `orientation` prop to control whether the separator is horizontal or vertical.  ```jsx &lt;Flex align="center" gap="4"&gt;   &lt;Separator orientation="horizontal" /&gt;    &lt;Separator orientation="vertical" /&gt; &lt;/Flex&gt; ```
