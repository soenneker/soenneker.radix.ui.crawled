		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Select  Displays a list of options for the user to pick from—triggered by a button.  ```jsx &lt;Select.Root defaultValue="apple"&gt;   &lt;Select.Trigger /&gt;    &lt;Select.Content&gt;     &lt;Select.Group&gt;       &lt;Select.Label&gt;Fruits&lt;/Select.Label&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;        &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="grape" disabled&gt;         Grape       &lt;/Select.Item&gt;     &lt;/Select.Group&gt;      &lt;Select.Separator /&gt;      &lt;Select.Group&gt;       &lt;Select.Label&gt;Vegetables&lt;/Select.Label&gt;        &lt;Select.Item value="carrot"&gt;Carrot&lt;/Select.Item&gt;        &lt;Select.Item value="potato"&gt;Potato&lt;/Select.Item&gt;     &lt;/Select.Group&gt;   &lt;/Select.Content&gt; &lt;/Select.Root&gt; ```  ## [API Reference](#api-reference)  ### [Root](#root)  Contains all the parts of a select. It inherits props from the Select primitive [Root](/primitives/docs/components/select#root) part.  | Prop   | Type                            | Default | | ------ | ------------------------------- | ------- | | `size` | `Responsive&lt;"1" \| "2" \| "3"&gt;` | `"2"`   |  ### [Trigger](#trigger)  The button that toggles the select. This component inherits props from the Select primitive [Trigger](/primitives/docs/components/select#trigger) and [Value](/primitives/docs/components/select#value) parts. It supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop          | Type                                                 | Default          | | ------------- | ---------------------------------------------------- | ---------------- | | `variant`     | `"classic" \| "surface" \| "soft" \| "ghost"`        | `"surface"`      | | `color`       | `enum`                                               | No default value | | `radius`      | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value | | `placeholder` | `string`                                             | No default value |  ### [Content](#content)  The component that pops out when the select is open. It inherits props from the [Select.Portal primitive](/primitives/docs/components/select#portal) and [Select.Content primitive](/primitives/docs/components/select#content) parts.  | Prop           | Type                | Default          | | -------------- | ------------------- | ---------------- | | `variant`      | `"solid" \| "soft"` | `"solid"`        | | `color`        | `enum`              | No default value | | `highContrast` | `boolean`           | No default value |  ### [Item](#item)  The component that contains the select items. It inherits props from the [Select.Item primitive](/primitives/docs/components/select#item) part.  ### [Group](#group)  Used to group multiple items. It inherits props from the [Select.Group primitive](/primitives/docs/components/select#group) part. Use in conjunction with `Select.Label` to ensure good accessibility via automatic labelling.  ### [Label](#label)  Used to render the label of a group, it isn't focusable using arrow keys. It inherits props from the [Select.Label primitive](/primitives/docs/components/select#label) part.  ### [Separator](#separator)  Used to visually separate items in the Select. It inherits props from the [Select.Separator primitive](/primitives/docs/components/select#separator) part.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex gap="3" align="center"&gt;   &lt;Select.Root size="1" defaultValue="apple"&gt;     &lt;Select.Trigger /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root size="2" defaultValue="apple"&gt;     &lt;Select.Trigger /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root size="3" defaultValue="apple"&gt;     &lt;Select.Trigger /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop on `Trigger` and `Content` to customize the visual style.  ```jsx &lt;Flex gap="3" align="center"&gt;   &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger variant="surface" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger variant="classic" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger variant="soft" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  #### Ghost  Use the `ghost` trigger variant to render the trigger without a visually containing element. Ghost triggers behave differently in layout as they use a negative margin to optically align themselves against their siblings while maintaining their padded active and hover states.  ```jsx &lt;Flex gap="3" align="center"&gt;   &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger variant="surface" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger variant="ghost" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop on `Trigger` and `Content` to assign a specific color value.  ```jsx &lt;Flex gap="3"&gt;   &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="indigo" variant="soft" /&gt;      &lt;Select.Content color="indigo"&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="cyan" variant="soft" /&gt;      &lt;Select.Content color="cyan"&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="orange" variant="soft" /&gt;      &lt;Select.Content color="orange"&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="crimson" variant="soft" /&gt;      &lt;Select.Content color="crimson"&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop on `Content` to increase item contrast.  ```jsx &lt;Flex gap="3"&gt;   &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="gray" /&gt;      &lt;Select.Content color="gray" variant="solid"&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger color="gray" /&gt;      &lt;Select.Content color="gray" variant="solid" highContrast&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex gap="3"&gt;   &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger radius="none" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger radius="large" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt;    &lt;Select.Root defaultValue="apple"&gt;     &lt;Select.Trigger radius="full" /&gt;      &lt;Select.Content&gt;       &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;     &lt;/Select.Content&gt;   &lt;/Select.Root&gt; &lt;/Flex&gt; ```  ### [Placeholder](#placeholder)  Use the `placeholder` prop to create a `Trigger` that doesn’t need an initial value.  ```jsx &lt;Select.Root&gt;   &lt;Select.Trigger placeholder="Pick a fruit" /&gt;    &lt;Select.Content&gt;     &lt;Select.Group&gt;       &lt;Select.Label&gt;Fruits&lt;/Select.Label&gt;        &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;        &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;        &lt;Select.Item value="grape" disabled&gt;         Grape       &lt;/Select.Item&gt;     &lt;/Select.Group&gt;      &lt;Select.Separator /&gt;      &lt;Select.Group&gt;       &lt;Select.Label&gt;Vegetables&lt;/Select.Label&gt;        &lt;Select.Item value="carrot"&gt;Carrot&lt;/Select.Item&gt;        &lt;Select.Item value="potato"&gt;Potato&lt;/Select.Item&gt;     &lt;/Select.Group&gt;   &lt;/Select.Content&gt; &lt;/Select.Root&gt; ```  ### [Position](#position)  Set `position="popper"` prop to position the select menu below the trigger.  ```jsx &lt;Select.Root defaultValue="apple"&gt;   &lt;Select.Trigger /&gt;    &lt;Select.Content position="popper"&gt;     &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;      &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;   &lt;/Select.Content&gt; &lt;/Select.Root&gt; ```  ### [With SSR](#with-ssr)  When using server-side rendering, you might notice a layout shift after hydration. This is because Trigger executes client-side code to display the selected item’s text. To avoid that layout shift, you can render it manually by mapping values.  ```jsx () =&gt; {   const data = {     apple: "Apple",      orange: "Orange",   };    const [value, setValue] = React.useState("apple");    return (     &lt;Select.Root value={value} onValueChange={setValue}&gt;       &lt;Select.Trigger&gt;{data[value]}&lt;/Select.Trigger&gt;        &lt;Select.Content&gt;         &lt;Select.Item value="apple"&gt;Apple&lt;/Select.Item&gt;          &lt;Select.Item value="orange"&gt;Orange&lt;/Select.Item&gt;       &lt;/Select.Content&gt;     &lt;/Select.Root&gt;   ); }; ```  ### [With an icon](#with-an-icon)  You can customise how Trigger renders the value by controlling its children manually. For example, you can render an icon next to the selected item’s text.  ```jsx () =&gt; {   const data = {     light: { label: "Light", icon: &lt;SunIcon /&gt; },      dark: { label: "Dark", icon: &lt;MoonIcon /&gt; },   };    const [value, setValue] = React.useState("light");    return (     &lt;Flex direction="column" maxWidth="160px"&gt;       &lt;Select.Root value={value} onValueChange={setValue}&gt;         &lt;Select.Trigger&gt;           &lt;Flex as="span" align="center" gap="2"&gt;             {data[value].icon}              {data[value].label}           &lt;/Flex&gt;         &lt;/Select.Trigger&gt;          &lt;Select.Content position="popper"&gt;           &lt;Select.Item value="light"&gt;Light&lt;/Select.Item&gt;            &lt;Select.Item value="dark"&gt;Dark&lt;/Select.Item&gt;         &lt;/Select.Content&gt;       &lt;/Select.Root&gt;     &lt;/Flex&gt;   ); }; ```
