		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Callout  Short message to attract user’s attention.  ```jsx &lt;Callout.Root&gt;   &lt;Callout.Icon&gt;     &lt;InfoCircledIcon /&gt;   &lt;/Callout.Icon&gt;    &lt;Callout.Text&gt;     You will need admin privileges to install and access this application.   &lt;/Callout.Text&gt; &lt;/Callout.Root&gt; ```  ## [API Reference](#api-reference)  ### [Root](#root)  Groups Icon and Text parts. This component is based on the `div` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                               | Default          | | -------------- | ---------------------------------- | ---------------- | | `asChild`      | `boolean`                          | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`    | `"2"`            | | `variant`      | `"soft" \| "surface" \| "outline"` | `"soft"`         | | `color`        | `enum`                             | No default value | | `highContrast` | `boolean`                          | No default value |  ### [Icon](#icon)  Provides width and height for the icon associated with the callout. This component is based on the `div` element.  ### [Text](#text)  Renders the callout text. This component is based on the `p` element.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex direction="column" gap="3" align="start"&gt;   &lt;Callout.Root size="1"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need admin privileges to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root size="2"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need admin privileges to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root size="3"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need admin privileges to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Callout.Root variant="soft"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need &lt;Link href="#"&gt;admin privileges&lt;/Link&gt; to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root variant="surface"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need &lt;Link href="#"&gt;admin privileges&lt;/Link&gt; to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root variant="outline"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need &lt;Link href="#"&gt;admin privileges&lt;/Link&gt; to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Callout.Root color="blue"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need &lt;Link href="#"&gt;admin privileges&lt;/Link&gt; to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root color="green"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       You will need &lt;Link href="#"&gt;admin privileges&lt;/Link&gt; to install and access this application.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root color="red"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt; ```  ### [High-contrast](#high-contrast)  Use the `HighContrast` prop to add additional contrast.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Callout.Root color="gray" variant="soft"&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       An update to Radix Themes is available. See what’s new in version 3.2.0.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt;    &lt;Callout.Root color="gray" variant="soft" highContrast&gt;     &lt;Callout.Icon&gt;       &lt;InfoCircledIcon /&gt;     &lt;/Callout.Icon&gt;      &lt;Callout.Text&gt;       An update to Radix Themes is available. See what’s new in version 3.2.0.     &lt;/Callout.Text&gt;   &lt;/Callout.Root&gt; &lt;/Flex&gt; ```  ### [As alert](#as-alert)  Add a native [WAI-ARIA `alert` role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/alert_role) to the callout when the user’s immediate attention is required, like when an error message appears. The screen reader will be interrupted, announcing the new content immediately.  ```jsx &lt;Callout.Root color="red" role="alert"&gt;   &lt;Callout.Icon&gt;     &lt;ExclamationTriangleIcon /&gt;   &lt;/Callout.Icon&gt;    &lt;Callout.Text&gt;     Access denied. Please contact the network administrator to view this page.   &lt;/Callout.Text&gt; &lt;/Callout.Root&gt; ```
