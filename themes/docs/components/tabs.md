		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Tabs  Set of content sections to be displayed one at a time.  ```jsx &lt;Tabs.Root defaultValue="account"&gt;   &lt;Tabs.List&gt;     &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;      &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;      &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;   &lt;/Tabs.List&gt;    &lt;Box pt="3"&gt;     &lt;Tabs.Content value="account"&gt;       &lt;Text size="2"&gt;Make changes to your account.&lt;/Text&gt;     &lt;/Tabs.Content&gt;      &lt;Tabs.Content value="documents"&gt;       &lt;Text size="2"&gt;Access and update your documents.&lt;/Text&gt;     &lt;/Tabs.Content&gt;      &lt;Tabs.Content value="settings"&gt;       &lt;Text size="2"&gt;Edit your profile or update contact information.&lt;/Text&gt;     &lt;/Tabs.Content&gt;   &lt;/Box&gt; &lt;/Tabs.Root&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Tabs primitive](/primitives/docs/components/tabs) and supports [common margin props](/themes/docs/overview/layout#margin-props).  ### [Root](#root)  Contains all component parts.  | Prop      | Type      | Default          | | --------- | --------- | ---------------- | | `asChild` | `boolean` | No default value |  ### [List](#list)  Contains the triggers that sit alongside the active content.  | Prop           | Type                                               | Default          | | -------------- | -------------------------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2"&gt;`                           | `"2"`            | | `wrap`         | `Responsive&lt;"nowrap" \| "wrap" \| "wrap-reverse"&gt;` | No default value | | `justify`      | `Responsive&lt;"start" \| "center" \| "end"&gt;`         | No default value | | `color`        | `enum`                                             | No default value | | `highContrast` | `boolean`                                          | No default value |  ### [Trigger](#trigger)  The button that activates its associated content.  ### [Content](#content)  Contains the content associated with each trigger.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the tab list.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List size="1"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt;    &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List size="2"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color) to the tab list.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="indigo"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt;    &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="cyan"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt;    &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="orange"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt;    &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="crimson"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="gray"&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt;    &lt;Tabs.Root defaultValue="account"&gt;     &lt;Tabs.List color="gray" highContrast&gt;       &lt;Tabs.Trigger value="account"&gt;Account&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="documents"&gt;Documents&lt;/Tabs.Trigger&gt;        &lt;Tabs.Trigger value="settings"&gt;Settings&lt;/Tabs.Trigger&gt;     &lt;/Tabs.List&gt;   &lt;/Tabs.Root&gt; &lt;/Flex&gt; ```  ### [Navigation](#navigation)  Tabs should not be used for page navigation. Use [Tab Nav](/themes/docs/components/tab-nav) instead, which is designed for this purpose and has equivalent styles.
