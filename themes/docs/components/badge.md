		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Badge  Stylized badge element.  ```jsx &lt;Flex gap="2"&gt;   &lt;Badge color="orange"&gt;In progress&lt;/Badge&gt;    &lt;Badge color="blue"&gt;In review&lt;/Badge&gt;    &lt;Badge color="green"&gt;Complete&lt;/Badge&gt; &lt;/Flex&gt; ```  ## [API Reference](#api-reference)  This component is based on the `span` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `asChild`      | `boolean`                                            | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"1"`            | | `variant`      | `"solid" \| "soft" \| "surface" \| "outline"`        | `"soft"`         | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex align="center" gap="2"&gt;   &lt;Badge size="1" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge size="2" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge size="3" color="indigo"&gt;     New   &lt;/Badge&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex gap="2"&gt;   &lt;Badge variant="solid" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge variant="soft" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge variant="surface" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge variant="outline" color="indigo"&gt;     New   &lt;/Badge&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="2"&gt;   &lt;Badge color="indigo"&gt;New&lt;/Badge&gt;    &lt;Badge color="cyan"&gt;New&lt;/Badge&gt;    &lt;Badge color="orange"&gt;New&lt;/Badge&gt;    &lt;Badge color="crimson"&gt;New&lt;/Badge&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column" gap="2"&gt;   &lt;Flex gap="2"&gt;     &lt;Badge color="gray" variant="solid"&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="soft"&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="surface"&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="outline"&gt;       New     &lt;/Badge&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Badge color="gray" variant="solid" highContrast&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="soft" highContrast&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="surface" highContrast&gt;       New     &lt;/Badge&gt;      &lt;Badge color="gray" variant="outline" highContrast&gt;       New     &lt;/Badge&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex gap="2"&gt;   &lt;Badge variant="solid" radius="none" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge variant="solid" radius="large" color="indigo"&gt;     New   &lt;/Badge&gt;    &lt;Badge variant="solid" radius="full" color="indigo"&gt;     New   &lt;/Badge&gt; &lt;/Flex&gt; ```
