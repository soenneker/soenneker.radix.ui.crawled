		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Text Field  Captures user input with an optional slot for buttons and icons.  ```jsx &lt;TextField.Root placeholder="Search the docs…"&gt;   &lt;TextField.Slot&gt;     &lt;MagnifyingGlassIcon height="16" width="16" /&gt;   &lt;/TextField.Slot&gt; &lt;/TextField.Root&gt; ```  ## [API Reference](#api-reference)  ### [Root](#root)  Groups Slot and Input parts. This component is based on the `div` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop      | Type                                                 | Default          | | --------- | ---------------------------------------------------- | ---------------- | | `size`    | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"2"`            | | `variant` | `"classic" \| "surface" \| "soft"`                   | `"surface"`      | | `color`   | `enum`                                               | No default value | | `radius`  | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value |  ### [Slot](#slot)  Contains icons or buttons associated with an Input.  | Prop    | Type                         | Default          | | ------- | ---------------------------- | ---------------- | | `side`  | `"left" \| "right"`          | No default value | | `color` | `enum`                       | No default value | | `gap`   | `Responsive&lt;enum \| string&gt;` | No default value | | `px`    | `Responsive&lt;enum \| string&gt;` | No default value | | `pl`    | `Responsive&lt;enum \| string&gt;` | No default value | | `pr`    | `Responsive&lt;enum \| string&gt;` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Box maxWidth="200px"&gt;     &lt;TextField.Root size="1" placeholder="Search the docs…" /&gt;   &lt;/Box&gt;    &lt;Box maxWidth="250px"&gt;     &lt;TextField.Root size="2" placeholder="Search the docs…" /&gt;   &lt;/Box&gt;    &lt;Box maxWidth="300px"&gt;     &lt;TextField.Root size="3" placeholder="Search the docs…" /&gt;   &lt;/Box&gt; &lt;/Flex&gt; ```  Use matching component sizes when composing Text Field with buttons. However, don’t use size 1 inputs with buttons—at this size, there is not enough vertical space to nest other interactive elements.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="400px"&gt;   &lt;Box maxWidth="200px"&gt;     &lt;TextField.Root placeholder="Search the docs…" size="1"&gt;       &lt;TextField.Slot&gt;         &lt;MagnifyingGlassIcon height="16" width="16" /&gt;       &lt;/TextField.Slot&gt;     &lt;/TextField.Root&gt;   &lt;/Box&gt;    &lt;Box maxWidth="250px"&gt;     &lt;TextField.Root placeholder="Search the docs…" size="2"&gt;       &lt;TextField.Slot&gt;         &lt;MagnifyingGlassIcon height="16" width="16" /&gt;       &lt;/TextField.Slot&gt;        &lt;TextField.Slot&gt;         &lt;IconButton size="1" variant="ghost"&gt;           &lt;DotsHorizontalIcon height="14" width="14" /&gt;         &lt;/IconButton&gt;       &lt;/TextField.Slot&gt;     &lt;/TextField.Root&gt;   &lt;/Box&gt;    &lt;Box maxWidth="300px"&gt;     &lt;TextField.Root placeholder="Search the docs…" size="3"&gt;       &lt;TextField.Slot&gt;         &lt;MagnifyingGlassIcon height="16" width="16" /&gt;       &lt;/TextField.Slot&gt;        &lt;TextField.Slot pr="3"&gt;         &lt;IconButton size="2" variant="ghost"&gt;           &lt;DotsHorizontalIcon height="16" width="16" /&gt;         &lt;/IconButton&gt;       &lt;/TextField.Slot&gt;     &lt;/TextField.Root&gt;   &lt;/Box&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextField.Root variant="surface" placeholder="Search the docs…" /&gt;    &lt;TextField.Root variant="classic" placeholder="Search the docs…" /&gt;    &lt;TextField.Root variant="soft" placeholder="Search the docs…" /&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextField.Root color="indigo" variant="soft" placeholder="Search the docs…" /&gt;    &lt;TextField.Root color="green" variant="soft" placeholder="Search the docs…" /&gt;    &lt;TextField.Root color="red" variant="soft" placeholder="Search the docs…" /&gt; &lt;/Flex&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextField.Root radius="none" placeholder="Search the docs…" /&gt;    &lt;TextField.Root radius="large" placeholder="Search the docs…" /&gt;    &lt;TextField.Root radius="full" placeholder="Search the docs…" /&gt; &lt;/Flex&gt; ```
