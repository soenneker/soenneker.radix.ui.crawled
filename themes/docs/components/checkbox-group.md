		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Checkbox Group  Set of interactive buttons where multiple options can be selected at a time.  ```jsx &lt;CheckboxGroup.Root defaultValue={["1"]} name="example"&gt;   &lt;CheckboxGroup.Item value="1"&gt;Fun&lt;/CheckboxGroup.Item&gt;    &lt;CheckboxGroup.Item value="2"&gt;Serious&lt;/CheckboxGroup.Item&gt;    &lt;CheckboxGroup.Item value="3"&gt;Smart&lt;/CheckboxGroup.Item&gt; &lt;/CheckboxGroup.Root&gt; ```  ## [API Reference](#api-reference)  This component is based on the `div` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  ### [Root](#root)  Contains all the parts of a checkbox group.  | Prop           | Type                               | Default          | | -------------- | ---------------------------------- | ---------------- | | `asChild`      | `boolean`                          | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`    | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"` | `"surface"`      | | `color`        | `enum`                             | No default value | | `highContrast` | `boolean`                          | No default value |  ### [Item](#item)  An item in the group that can be checked.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the checkbox size.  ```jsx &lt;Flex align="center" gap="2"&gt;   &lt;CheckboxGroup.Root size="1" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root size="2" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root size="3" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the checkboxes.  ```jsx &lt;Flex gap="2"&gt;   &lt;Flex direction="column" asChild gap="2"&gt;     &lt;CheckboxGroup.Root variant="surface" defaultValue="1"&gt;       &lt;CheckboxGroup.Item value="1" /&gt;        &lt;CheckboxGroup.Item value="2" /&gt;     &lt;/CheckboxGroup.Root&gt;   &lt;/Flex&gt;    &lt;Flex direction="column" asChild gap="2"&gt;     &lt;CheckboxGroup.Root variant="classic" defaultValue="1"&gt;       &lt;CheckboxGroup.Item value="1" /&gt;        &lt;CheckboxGroup.Item value="2" /&gt;     &lt;/CheckboxGroup.Root&gt;   &lt;/Flex&gt;    &lt;Flex direction="column" asChild gap="2"&gt;     &lt;CheckboxGroup.Root variant="soft" defaultValue="1"&gt;       &lt;CheckboxGroup.Item value="1" /&gt;        &lt;CheckboxGroup.Item value="2" /&gt;     &lt;/CheckboxGroup.Root&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="2"&gt;   &lt;CheckboxGroup.Root color="indigo" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="cyan" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="orange" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="crimson" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Grid rows="2" gap="2" display="inline-grid" flow="column"&gt;   &lt;CheckboxGroup.Root color="indigo" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="indigo" defaultValue="1" highContrast&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="cyan" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="cyan" defaultValue="1" highContrast&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="orange" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="orange" defaultValue="1" highContrast&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="crimson" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="crimson" defaultValue="1" highContrast&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="gray" defaultValue="1"&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root color="gray" defaultValue="1" highContrast&gt;     &lt;CheckboxGroup.Item value="1" /&gt;   &lt;/CheckboxGroup.Root&gt; &lt;/Grid&gt; ```  ### [Alignment](#alignment)  Composing `CheckboxGroup.Item` within `Text` automatically centers it with the first line of text.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;CheckboxGroup.Root size="1" defaultValue="1"&gt;     &lt;Text as="label" size="2"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="1" /&gt; Default       &lt;/Flex&gt;     &lt;/Text&gt;      &lt;Text as="label" size="2"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="2" /&gt; Compact       &lt;/Flex&gt;     &lt;/Text&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root size="2" defaultValue="1"&gt;     &lt;Text as="label" size="3"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="1" /&gt; Default       &lt;/Flex&gt;     &lt;/Text&gt;      &lt;Text as="label" size="3"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="2" /&gt; Compact       &lt;/Flex&gt;     &lt;/Text&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root size="3" defaultValue="1"&gt;     &lt;Text as="label" size="4"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="1" /&gt; Default       &lt;/Flex&gt;     &lt;/Text&gt;      &lt;Text as="label" size="4"&gt;       &lt;Flex gap="2"&gt;         &lt;CheckboxGroup.Item value="2" /&gt; Compact       &lt;/Flex&gt;     &lt;/Text&gt;   &lt;/CheckboxGroup.Root&gt; &lt;/Flex&gt; ```  It is automatically well-aligned with multi-line text too.  ### [Disabled](#disabled)  Use the native `disabled` attribute to create a disabled checkbox.  ```jsx &lt;Flex direction="column" gap="2"&gt;   &lt;CheckboxGroup.Root defaultValue="2"&gt;     &lt;CheckboxGroup.Item value="1"&gt;Off&lt;/CheckboxGroup.Item&gt;      &lt;CheckboxGroup.Item value="2"&gt;On&lt;/CheckboxGroup.Item&gt;   &lt;/CheckboxGroup.Root&gt;    &lt;CheckboxGroup.Root defaultValue="2"&gt;     &lt;CheckboxGroup.Item value="1" disabled&gt;       Off     &lt;/CheckboxGroup.Item&gt;      &lt;CheckboxGroup.Item value="2" disabled&gt;       On     &lt;/CheckboxGroup.Item&gt;   &lt;/CheckboxGroup.Root&gt; &lt;/Flex&gt; ```
