		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Link  Semantic element for navigation between pages.  ```jsx &lt;Link href="#"&gt;Sign up&lt;/Link&gt; ```  ## [API Reference](#api-reference)  This component is based on the `a` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                      | Default          | | -------------- | --------------------------------------------------------- | ---------------- | | `asChild`      | `boolean`                                                 | No default value | | `size`         | `Responsive&lt;enum&gt;`                                        | No default value | | `weight`       | `Responsive&lt;"light" \| "regular" \| "medium" \| "bold"&gt;`  | No default value | | `trim`         | `Responsive&lt;"normal" \| "start" \| "end" \| "both"&gt;`      | No default value | | `truncate`     | `boolean`                                                 | No default value | | `wrap`         | `Responsive&lt;"wrap" \| "nowrap" \| "pretty" \| "balance"&gt;` | No default value | | `underline`    | `"auto" \| "always" \| "hover" \| "none"`                 | `"auto"`         | | `color`        | `enum`                                                    | No default value | | `highContrast` | `boolean`                                                 | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the link. The prop also provides correct line height and corrective letter spacing—as text size increases, the relative line height and letter spacing decrease.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Link href="#" size="1"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="2"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="3"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="4"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="5"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="6"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="7"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="8"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" size="9"&gt;     Sign up   &lt;/Link&gt; &lt;/Flex&gt; ```  ### [Weight](#weight)  Use the `weight` prop to set the text weight.  ```jsx &lt;Flex direction="column"&gt;   &lt;Link href="#" weight="regular"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" weight="medium"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" weight="bold"&gt;     Sign up   &lt;/Link&gt; &lt;/Flex&gt; ```  ### [Truncate](#truncate)  Use the `truncate` prop to truncate text with an ellipsis when it overflows its container.  ```jsx &lt;Flex maxWidth="150px"&gt;   &lt;Link href="#" truncate&gt;     Sign up to the newsletter   &lt;/Link&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column"&gt;   &lt;Link href="#" color="indigo"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" color="cyan"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" color="orange"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" color="crimson"&gt;     Sign up   &lt;/Link&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column"&gt;   &lt;Link href="#" color="gray"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" color="gray" highContrast&gt;     Sign up   &lt;/Link&gt; &lt;/Flex&gt; ```  ### [Underline](#underline)  Use the `underline` prop to manage the visibility of the underline affordance.  ```jsx &lt;Flex direction="column"&gt;   &lt;Link href="#" underline="hover"&gt;     Sign up   &lt;/Link&gt;    &lt;Link href="#" underline="always"&gt;     Sign up   &lt;/Link&gt; &lt;/Flex&gt; ```
