		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Text Area  Captures multi-line user input.  ```jsx &lt;TextArea placeholder="Reply to comment…" /&gt; ```  ## [API Reference](#api-reference)  This component is based on the `textarea` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop      | Type                                                         | Default          | | --------- | ------------------------------------------------------------ | ---------------- | | `size`    | `Responsive&lt;"1" \| "2" \| "3"&gt;`                              | `"2"`            | | `variant` | `"classic" \| "surface" \| "soft"`                           | `"surface"`      | | `resize`  | `Responsive&lt;"none" \| "vertical" \| "horizontal" \| "both"&gt;` | No default value | | `color`   | `enum`                                                       | No default value | | `radius`  | `"none" \| "small" \| "medium" \| "large" \| "full"`         | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Box maxWidth="200px"&gt;     &lt;TextArea size="1" placeholder="Reply to comment…" /&gt;   &lt;/Box&gt;    &lt;Box maxWidth="250px"&gt;     &lt;TextArea size="2" placeholder="Reply to comment…" /&gt;   &lt;/Box&gt;    &lt;Box maxWidth="300px"&gt;     &lt;TextArea size="3" placeholder="Reply to comment…" /&gt;   &lt;/Box&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextArea variant="surface" placeholder="Reply to comment…" /&gt;    &lt;TextArea variant="classic" placeholder="Reply to comment…" /&gt;    &lt;TextArea variant="soft" placeholder="Reply to comment…" /&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextArea color="blue" variant="soft" placeholder="Reply to comment…" /&gt;    &lt;TextArea color="green" variant="soft" placeholder="Reply to comment…" /&gt;    &lt;TextArea color="red" variant="soft" placeholder="Reply to comment…" /&gt; &lt;/Flex&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextArea radius="none" placeholder="Search the docs…" /&gt;    &lt;TextArea radius="large" placeholder="Search the docs…" /&gt;    &lt;TextArea radius="full" placeholder="Search the docs…" /&gt; &lt;/Flex&gt; ```  ### [Resize](#resize)  Use the `resize` prop to enable resizing on one or both axis.  ```jsx &lt;Flex direction="column" gap="3" maxWidth="250px"&gt;   &lt;TextArea resize="none" placeholder="Search the docs…" /&gt;    &lt;TextArea resize="vertical" placeholder="Search the docs…" /&gt;    &lt;TextArea resize="horizontal" placeholder="Search the docs…" /&gt;    &lt;TextArea resize="both" placeholder="Search the docs…" /&gt; &lt;/Flex&gt; ```
