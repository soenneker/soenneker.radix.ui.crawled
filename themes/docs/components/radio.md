		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Radio  Standalone radio button that can be used in any layout.  ```jsx &lt;Flex align="start" direction="column" gap="1"&gt;   &lt;Flex asChild gap="2"&gt;     &lt;Text as="label" size="2"&gt;       &lt;Radio name="example" value="1" defaultChecked /&gt;       Default     &lt;/Text&gt;   &lt;/Flex&gt;    &lt;Flex asChild gap="2"&gt;     &lt;Text as="label" size="2"&gt;       &lt;Radio name="example" value="2" /&gt;       Comfortable     &lt;/Text&gt;   &lt;/Flex&gt;    &lt;Flex asChild gap="2"&gt;     &lt;Text as="label" size="2"&gt;       &lt;Radio name="example" value="3" /&gt;       Compact     &lt;/Text&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Radio Group primitive](/primitives/docs/components/radio-group) and supports [common margin props](/themes/docs/overview/layout#margin-props).  ### [Root](#root)  Contains all the parts of a radio group.  | Prop           | Type                               | Default          | | -------------- | ---------------------------------- | ---------------- | | `asChild`      | `boolean`                          | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`    | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"` | `"surface"`      | | `color`        | `enum`                             | No default value | | `highContrast` | `boolean`                          | No default value |  ### [Item](#item)  An item in the group that can be checked.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the radio button size.  ```jsx &lt;Flex align="center" gap="4"&gt;   &lt;Flex gap="2"&gt;     &lt;Radio size="1" name="size-1" value="1" defaultChecked /&gt;      &lt;Radio size="1" name="size-1" value="2" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Radio size="2" name="size-2" value="1" defaultChecked /&gt;      &lt;Radio size="2" name="size-2" value="2" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Radio size="3" name="size-3" value="1" defaultChecked /&gt;      &lt;Radio size="3" name="size-3" value="2" /&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the radio buttons.  ```jsx &lt;Flex align="center" gap="4"&gt;   &lt;Flex gap="2"&gt;     &lt;Radio variant="surface" name="surface" value="1" defaultChecked /&gt;      &lt;Radio variant="surface" name="surface" value="2" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Radio variant="classic" name="classic" value="1" defaultChecked /&gt;      &lt;Radio variant="classic" name="classic" value="2" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Radio variant="soft" name="soft" value="1" defaultChecked /&gt;      &lt;Radio variant="soft" name="soft" value="2" /&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex as="span" gap="2"&gt;   &lt;Radio color="indigo" defaultChecked /&gt;    &lt;Radio color="cyan" defaultChecked /&gt;    &lt;Radio color="orange" defaultChecked /&gt;    &lt;Radio color="crimson" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Grid columns="5" display="inline-grid" gap="2"&gt;   &lt;Radio color="indigo" defaultChecked /&gt;    &lt;Radio color="cyan" defaultChecked /&gt;    &lt;Radio color="orange" defaultChecked /&gt;    &lt;Radio color="crimson" defaultChecked /&gt;    &lt;Radio color="gray" defaultChecked /&gt;    &lt;Radio color="indigo" defaultChecked highContrast /&gt;    &lt;Radio color="cyan" defaultChecked highContrast /&gt;    &lt;Radio color="orange" defaultChecked highContrast /&gt;    &lt;Radio color="crimson" defaultChecked highContrast /&gt;    &lt;Radio color="gray" defaultChecked highContrast /&gt; &lt;/Grid&gt; ```  ### [Alignment](#alignment)  Composing `Radio` within `Text` automatically centers it with the first line of text. It is automatically well-aligned with multi-line text too.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Flex align="start" direction="column" gap="1"&gt;     &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2"&gt;         &lt;Radio size="1" name="alignment-1" value="1" defaultChecked /&gt;         Default       &lt;/Text&gt;     &lt;/Flex&gt;      &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2"&gt;         &lt;Radio size="1" name="alignment-1" value="2" /&gt;         Compact       &lt;/Text&gt;     &lt;/Flex&gt;   &lt;/Flex&gt;    &lt;Flex align="start" direction="column" gap="1"&gt;     &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="3"&gt;         &lt;Radio size="2" name="alignment-2" value="1" defaultChecked /&gt;         Default       &lt;/Text&gt;     &lt;/Flex&gt;      &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="3"&gt;         &lt;Radio size="2" name="alignment-2" value="2" /&gt;         Compact       &lt;/Text&gt;     &lt;/Flex&gt;   &lt;/Flex&gt;    &lt;Flex align="start" direction="column" gap="1"&gt;     &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="4"&gt;         &lt;Radio size="3" name="alignment-3" value="1" defaultChecked /&gt;         Default       &lt;/Text&gt;     &lt;/Flex&gt;      &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="4"&gt;         &lt;Radio size="3" name="alignment-3" value="2" /&gt;         Compact       &lt;/Text&gt;     &lt;/Flex&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Disabled](#disabled)  Use the native `disabled` attribute to create a disabled radio button.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Flex align="start" direction="column" gap="1"&gt;     &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2"&gt;         &lt;Radio name="enabled" value="1" defaultChecked /&gt;         On       &lt;/Text&gt;     &lt;/Flex&gt;      &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2"&gt;         &lt;Radio name="enabled" value="2" /&gt;         Off       &lt;/Text&gt;     &lt;/Flex&gt;   &lt;/Flex&gt;    &lt;Flex align="start" direction="column" gap="1"&gt;     &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2" color="gray"&gt;         &lt;Radio disabled name="disabled" value="1" defaultChecked /&gt;         On       &lt;/Text&gt;     &lt;/Flex&gt;      &lt;Flex asChild gap="2"&gt;       &lt;Text as="label" size="2" color="gray"&gt;         &lt;Radio disabled name="disabled" value="2" /&gt;         Off       &lt;/Text&gt;     &lt;/Flex&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```
