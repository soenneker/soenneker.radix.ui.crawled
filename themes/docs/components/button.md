		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Button  Trigger an action or event, such as submitting a form or displaying a dialog.  ```jsx &lt;Button&gt;   &lt;BookmarkIcon /&gt; Bookmark &lt;/Button&gt; ```  ## [API Reference](#api-reference)  This component is based on the `button` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `asChild`      | `boolean`                                            | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;`               | `"2"`            | | `variant`      | `enum`                                               | `"solid"`        | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value | | `loading`      | `boolean`                                            | `false`          |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the button.  ```jsx &lt;Flex gap="3" align="center"&gt;   &lt;Button size="1" variant="soft"&gt;     Edit profile   &lt;/Button&gt;    &lt;Button size="2" variant="soft"&gt;     Edit profile   &lt;/Button&gt;    &lt;Button size="3" variant="soft"&gt;     Edit profile   &lt;/Button&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the button.  ```jsx &lt;Flex align="center" gap="3"&gt;   &lt;Button variant="classic"&gt;Edit profile&lt;/Button&gt;    &lt;Button variant="solid"&gt;Edit profile&lt;/Button&gt;    &lt;Button variant="soft"&gt;Edit profile&lt;/Button&gt;    &lt;Button variant="surface"&gt;Edit profile&lt;/Button&gt;    &lt;Button variant="outline"&gt;Edit profile&lt;/Button&gt; &lt;/Flex&gt; ```  #### Ghost  Use the `ghost` variant to display a button without chrome. Ghost buttons behave like text in layout, as they use a negative margin to optically align themselves against their siblings while maintaining the padding in active and hover states.  ```jsx &lt;Button variant="ghost"&gt;Edit profile&lt;/Button&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="3"&gt;   &lt;Button color="indigo" variant="soft"&gt;     Edit profile   &lt;/Button&gt;    &lt;Button color="cyan" variant="soft"&gt;     Edit profile   &lt;/Button&gt;    &lt;Button color="orange" variant="soft"&gt;     Edit profile   &lt;/Button&gt;    &lt;Button color="crimson" variant="soft"&gt;     Edit profile   &lt;/Button&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column" gap="3"&gt;       Edit profile     &lt;/Button&gt;      &lt;Button color="gray" variant="soft" highContrast&gt;       Edit profile     &lt;/Button&gt;      &lt;Button color="gray" variant="surface" highContrast&gt;     Edit profile   &lt;/Button&gt; &lt;/Flex&gt; ```  ### [With icons](#with-icons)  You can nest icons directly inside the button. An appropriate gap is provided automatically.  ```jsx &lt;Flex gap="3"&gt;   &lt;Button variant="classic"&gt;     &lt;BookmarkIcon /&gt; Bookmark   &lt;/Button&gt;    &lt;Button variant="solid"&gt;     &lt;BookmarkIcon /&gt; Bookmark   &lt;/Button&gt;    &lt;Button variant="soft"&gt;     &lt;BookmarkIcon /&gt; Bookmark   &lt;/Button&gt;    &lt;Button variant="surface"&gt;     &lt;BookmarkIcon /&gt; Bookmark   &lt;/Button&gt;    &lt;Button variant="outline"&gt;     &lt;BookmarkIcon /&gt; Bookmark   &lt;/Button&gt; &lt;/Flex&gt; ```  ### [Loading](#loading)  Use the `loading` prop to display a loading spinner in place of button content, preserving the original size of the button in its normal state. The button will be disabled while loading.  ```jsx &lt;Flex gap="3"&gt;   &lt;Button loading variant="classic"&gt;     Bookmark   &lt;/Button&gt;    &lt;Button loading variant="solid"&gt;     Bookmark   &lt;/Button&gt;    &lt;Button loading variant="soft"&gt;     Bookmark   &lt;/Button&gt;    &lt;Button loading variant="surface"&gt;     Bookmark   &lt;/Button&gt;    &lt;Button loading variant="outline"&gt;     Bookmark   &lt;/Button&gt; &lt;/Flex&gt; ```  If you have an icon inside the button, you can use the button’s `disabled` state and wrap the icon in a standalone [Spinner](/themes/docs/components/spinner) to achieve a more sophisticated design.  ```jsx &lt;Flex gap="3"&gt;   &lt;Button disabled variant="classic"&gt;     &lt;Spinner loading&gt;       &lt;BookmarkIcon /&gt;     &lt;/Spinner&gt;     Bookmark   &lt;/Button&gt;    &lt;Button disabled variant="solid"&gt;     &lt;Spinner loading&gt;       &lt;BookmarkIcon /&gt;     &lt;/Spinner&gt;     Bookmark   &lt;/Button&gt;    &lt;Button disabled variant="soft"&gt;     &lt;Spinner loading&gt;       &lt;BookmarkIcon /&gt;     &lt;/Spinner&gt;     Bookmark   &lt;/Button&gt;    &lt;Button disabled variant="surface"&gt;     &lt;Spinner loading&gt;       &lt;BookmarkIcon /&gt;     &lt;/Spinner&gt;     Bookmark   &lt;/Button&gt;    &lt;Button disabled variant="outline"&gt;     &lt;Spinner loading&gt;       &lt;BookmarkIcon /&gt;     &lt;/Spinner&gt;     Bookmark   &lt;/Button&gt; &lt;/Flex&gt; ```
