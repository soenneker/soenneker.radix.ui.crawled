		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Progress  Displays a progress bar related to a task.  ```jsx &lt;Box maxWidth="300px"&gt;   &lt;Progress /&gt; &lt;/Box&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Progress primitive](/primitives/docs/components/progress) and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"`                   | `"surface"`      | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value | | `duration`     | `string`                                             | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Progress value={25} size="1" /&gt;    &lt;Progress value={50} size="2" /&gt;    &lt;Progress value={75} size="3" /&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Progress value={25} variant="classic" /&gt;    &lt;Progress value={50} variant="surface" /&gt;    &lt;Progress value={75} variant="soft" /&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Progress value={20} color="indigo" /&gt;    &lt;Progress value={40} color="cyan" /&gt;    &lt;Progress value={60} color="orange" /&gt;    &lt;Progress value={80} color="crimson" /&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Grid columns="2" gap="4"&gt;   &lt;Progress value={10} color="indigo" /&gt;    &lt;Progress value={10} color="indigo" highContrast /&gt;    &lt;Progress value={30} color="cyan" /&gt;    &lt;Progress value={30} color="cyan" highContrast /&gt;    &lt;Progress value={50} color="orange" /&gt;    &lt;Progress value={50} color="orange" highContrast /&gt;    &lt;Progress value={70} color="crimson" /&gt;    &lt;Progress value={70} color="crimson" highContrast /&gt;    &lt;Progress value={90} color="gray" /&gt;    &lt;Progress value={90} color="gray" highContrast /&gt; &lt;/Grid&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Progress value={25} radius="none" /&gt;    &lt;Progress value={50} radius="small" /&gt;    &lt;Progress value={75} radius="full" /&gt; &lt;/Flex&gt; ```  ### [With controlled value](#with-controlled-value)  Use the `value` prop to provide a precise indication of the task progress.  ```jsx &lt;Progress value={75} /&gt; ```  ### [With custom duration](#with-custom-duration)  Use the `duration` prop to indicate an approximate duration of an indeterminate task. Once the duration times out, the progress bar will start an indeterminate animation.  ```jsx &lt;Progress duration="30s" /&gt; ```  When an approximate duration can be estimated, the Progress component is still useful over [Spinner](/themes/docs/components/spinner), which doesnâ€™t provide any visual cues towards the progress of the task.
