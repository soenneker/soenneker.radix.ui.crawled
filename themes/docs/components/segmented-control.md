		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Segmented Control  Toggle buttons for switching between different values or views.  ```jsx &lt;SegmentedControl.Root defaultValue="inbox"&gt;   &lt;SegmentedControl.Item value="inbox"&gt;Inbox&lt;/SegmentedControl.Item&gt;    &lt;SegmentedControl.Item value="drafts"&gt;Drafts&lt;/SegmentedControl.Item&gt;    &lt;SegmentedControl.Item value="sent"&gt;Sent&lt;/SegmentedControl.Item&gt; &lt;/SegmentedControl.Root&gt; ```  ## [API Reference](#api-reference)  This component inherits props and functionality from the [Toggle Group primitive](/primitives/docs/components/toggle-group). It supports [common margin props](/themes/docs/overview/layout#margin-props).  ### [Root](#root)  Contains the items of the control.  | Prop       | Type                                                 | Default          | | ---------- | ---------------------------------------------------- | ---------------- | | `disabled` | `boolean`                                            | `false`          | | `size`     | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"2"`            | | `variant`  | `"surface" \| "classic"`                             | `"surface"`      | | `radius`   | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value |  ### [Item](#item)  Represents individual values of the control.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the control.  ```jsx &lt;Flex align="start" direction="column" gap="4"&gt;    &lt;SegmentedControl.Root defaultValue="inbox" size="3"&gt;     &lt;SegmentedControl.Item value="inbox"&gt;Inbox&lt;/SegmentedControl.Item&gt;      &lt;SegmentedControl.Item value="drafts"&gt;Drafts&lt;/SegmentedControl.Item&gt;
