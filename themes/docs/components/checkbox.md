		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Checkbox  Base input element to toggle an option on and off.  ```jsx &lt;Text as="label" size="2"&gt;   &lt;Flex gap="2"&gt;     &lt;Checkbox defaultChecked /&gt;     Agree to Terms and Conditions   &lt;/Flex&gt; &lt;/Text&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Checkbox primitive](/primitives/docs/components/checkbox) and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                               | Default          | | -------------- | ---------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`    | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"` | `"surface"`      | | `color`        | `enum`                             | No default value | | `highContrast` | `boolean`                          | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the checkbox.  ```jsx &lt;Flex align="center" gap="2"&gt;   &lt;Checkbox size="1" defaultChecked /&gt;    &lt;Checkbox size="2" defaultChecked /&gt;    &lt;Checkbox size="3" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the checkbox.  ```jsx &lt;Flex align="center" gap="4"&gt;   &lt;Flex gap="2"&gt;     &lt;Checkbox variant="surface" defaultChecked /&gt;      &lt;Checkbox variant="surface" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Checkbox variant="classic" defaultChecked /&gt;      &lt;Checkbox variant="classic" /&gt;   &lt;/Flex&gt;    &lt;Flex gap="2"&gt;     &lt;Checkbox variant="soft" defaultChecked /&gt;      &lt;Checkbox variant="soft" /&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="2"&gt;   &lt;Checkbox color="indigo" defaultChecked /&gt;    &lt;Checkbox color="cyan" defaultChecked /&gt;    &lt;Checkbox color="orange" defaultChecked /&gt;    &lt;Checkbox color="crimson" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Grid columns="5" display="inline-grid" gap="2"&gt;   &lt;Checkbox color="indigo" defaultChecked /&gt;    &lt;Checkbox color="cyan" defaultChecked /&gt;    &lt;Checkbox color="orange" defaultChecked /&gt;    &lt;Checkbox color="crimson" defaultChecked /&gt;    &lt;Checkbox color="gray" defaultChecked /&gt;    &lt;Checkbox color="indigo" defaultChecked highContrast /&gt;    &lt;Checkbox color="cyan" defaultChecked highContrast /&gt;    &lt;Checkbox color="orange" defaultChecked highContrast /&gt;    &lt;Checkbox color="crimson" defaultChecked highContrast /&gt;    &lt;Checkbox color="gray" defaultChecked highContrast /&gt; &lt;/Grid&gt; ```  ### [Alignment](#alignment)  Composing `Checkbox` within `Text` automatically centers it with the first line of text.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Text as="label" size="2"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox size="1" defaultChecked /&gt; Agree to Terms and Conditions     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="3"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox size="2" defaultChecked /&gt; Agree to Terms and Conditions     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="4"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox size="3" defaultChecked /&gt; Agree to Terms and Conditions     &lt;/Flex&gt;   &lt;/Text&gt; &lt;/Flex&gt; ```  It is automatically well-aligned with multi-line text too.  ```jsx &lt;Box maxWidth="300px"&gt;   &lt;Text as="label" size="3"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox defaultChecked /&gt; I understand that these documents are confidential and cannot be       shared with a third party.     &lt;/Flex&gt;   &lt;/Text&gt; &lt;/Box&gt; ```  ### [Disabled](#disabled)  Use the native `disabled` attribute to create a disabled checkbox.  ```jsx &lt;Flex direction="column" gap="2"&gt;   &lt;Text as="label" size="2"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox /&gt;       Not checked     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox defaultChecked /&gt;       Checked     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2" color="gray"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox disabled /&gt;       Not checked     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2" color="gray"&gt;     &lt;Flex as="span" gap="2"&gt;       &lt;Checkbox disabled defaultChecked /&gt;       Checked     &lt;/Flex&gt;   &lt;/Text&gt; &lt;/Flex&gt; ```  ### [Indeterminate](#indeterminate)  Use the `"indeterminate"` value to create an indeterminate checkbox.  ```jsx &lt;Flex gap="2"&gt;   &lt;Checkbox defaultChecked="indeterminate" /&gt;    &lt;Checkbox checked="indeterminate" /&gt; &lt;/Flex&gt; ```
