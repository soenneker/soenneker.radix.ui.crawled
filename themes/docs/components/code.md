		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Code  Marks text to signify a short fragment of computer code.  ```jsx &lt;Code&gt;console.log()&lt;/Code&gt; ```  ## [API Reference](#api-reference)  This component is based on the `code` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                      | Default          | | -------------- | --------------------------------------------------------- | ---------------- | | `asChild`      | `boolean`                                                 | No default value | | `size`         | `Responsive&lt;enum&gt;`                                        | No default value | | `variant`      | `"solid" \| "soft" \| "outline" \| "ghost"`               | `"soft"`         | | `weight`       | `Responsive&lt;"light" \| "regular" \| "medium" \| "bold"&gt;`  | No default value | | `color`        | `enum`                                                    | No default value | | `highContrast` | `boolean`                                                 | No default value | | `truncate`     | `boolean`                                                 | No default value | | `wrap`         | `Responsive&lt;"wrap" \| "nowrap" \| "pretty" \| "balance"&gt;` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control text size. This prop also provides correct line height and corrective letter spacing—as text size increases, the relative line height and letter spacing decrease.  ```jsx &lt;Flex direction="column" gap="3" align="start"&gt;   &lt;Code size="1"&gt;console.log()&lt;/Code&gt;    &lt;Code size="2"&gt;console.log()&lt;/Code&gt;    &lt;Code size="3"&gt;console.log()&lt;/Code&gt;    &lt;Code size="4"&gt;console.log()&lt;/Code&gt;    &lt;Code size="5"&gt;console.log()&lt;/Code&gt;    &lt;Code size="6"&gt;console.log()&lt;/Code&gt;    &lt;Code size="7"&gt;console.log()&lt;/Code&gt;    &lt;Code size="8"&gt;console.log()&lt;/Code&gt;    &lt;Code size="9"&gt;console.log()&lt;/Code&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" align="start" gap="2"&gt;   &lt;Code variant="solid"&gt;console.log()&lt;/Code&gt;    &lt;Code variant="soft"&gt;console.log()&lt;/Code&gt;    &lt;Code variant="outline"&gt;console.log()&lt;/Code&gt;    &lt;Code variant="ghost"&gt;console.log()&lt;/Code&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" align="start" gap="2"&gt;   &lt;Code color="indigo"&gt;console.log()&lt;/Code&gt;    &lt;Code color="crimson"&gt;console.log()&lt;/Code&gt;    &lt;Code color="cyan"&gt;console.log()&lt;/Code&gt;    &lt;Code color="orange"&gt;console.log()&lt;/Code&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex gap="3"&gt;   &lt;Flex direction="column" align="start" gap="2"&gt;     &lt;Code color="gray" variant="solid"&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="soft"&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="outline"&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="ghost"&gt;       console.log()     &lt;/Code&gt;   &lt;/Flex&gt;    &lt;Flex direction="column" align="start" gap="2"&gt;     &lt;Code color="gray" variant="solid" highContrast&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="soft" highContrast&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="outline" highContrast&gt;       console.log()     &lt;/Code&gt;      &lt;Code color="gray" variant="ghost" highContrast&gt;       console.log()     &lt;/Code&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Weight](#weight)  Use the `weight` prop to set the text weight.  ```jsx &lt;Flex direction="column" gap="2" align="start"&gt;   &lt;Code weight="regular"&gt;console.log()&lt;/Code&gt;    &lt;Code weight="bold"&gt;console.log()&lt;/Code&gt; &lt;/Flex&gt; ```  ### [Truncate](#truncate)  Use the `truncate` prop to truncate text with an ellipsis when it overflows its container.  ```jsx &lt;Flex maxWidth="200px"&gt;   &lt;Code truncate&gt;linear-gradient(red, orange, yellow, green, blue);&lt;/Code&gt; &lt;/Flex&gt; ```
