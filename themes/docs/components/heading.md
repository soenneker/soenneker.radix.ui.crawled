		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Heading  Semantic heading element.  ```jsx &lt;Heading&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; ```  ## [API Reference](#api-reference)  This component is based on the `h1` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                      | Default          | | -------------- | --------------------------------------------------------- | ---------------- | | `as`           | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6"`            | `"h1"`           | | `asChild`      | `boolean`                                                 | No default value | | `size`         | `Responsive&lt;enum&gt;`                                        | `"6"`            | | `weight`       | `Responsive&lt;"light" \| "regular" \| "medium" \| "bold"&gt;`  | No default value | | `align`        | `Responsive&lt;"left" \| "center" \| "right"&gt;`               | No default value | | `trim`         | `Responsive&lt;"normal" \| "start" \| "end" \| "both"&gt;`      | No default value | | `truncate`     | `boolean`                                                 | No default value | | `wrap`         | `Responsive&lt;"wrap" \| "nowrap" \| "pretty" \| "balance"&gt;` | No default value | | `color`        | `enum`                                                    | No default value | | `highContrast` | `boolean`                                                 | No default value |  ## [Examples](#examples)  ### [As another element](#as-another-element)  Use the `as` prop to change the heading level. This prop is purely semantic and does not change the visual appearance.  ```jsx &lt;Heading as="h1"&gt;Level 1&lt;/Heading&gt;  &lt;Heading as="h2"&gt;Level 2&lt;/Heading&gt;  &lt;Heading as="h3"&gt;Level 3&lt;/Heading&gt; ```  ### [Size](#size)  Use the `size` prop to control the size of the heading. The prop also provides correct line height and corrective letter spacing—as text size increases, the relative line height and letter spacing decrease.  Heading sizes match [Text](/themes/components/text) sizes. However, the line heights are set a bit tighter as headings tend to be shorter than running text.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Heading size="1"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="2"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="3"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="4"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="5"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="6"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="7"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="8"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading size="9"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  ### [Weight](#weight)  Use the `weight` prop to set the text weight.  ```jsx &lt;Heading weight="regular"&gt;    The quick brown fox jumps over the lazy dog.  &lt;/Heading&gt;    &lt;Heading weight="medium"&gt;    The quick brown fox jumps over the lazy dog.  &lt;/Heading&gt;    &lt;Heading weight="bold"&gt;    The quick brown fox jumps over the lazy dog.  &lt;/Heading&gt; ```  ### [Align](#align)  Use the `align` prop to set text alignment.  ```jsx &lt;Heading align="left"&gt;Left-aligned&lt;/Heading&gt;  &lt;Heading align="center"&gt;Center-aligned&lt;/Heading&gt;  &lt;Heading align="right"&gt;Right-aligned&lt;/Heading&gt; ```  ### [Trim](#trim)  Use the `trim` prop to trim the leading space at the start, end, or both sides of the text box.  The prop works similarly to the upcoming [half-leading control](https://www.w3.org/TR/css-inline-3/#leading-trim) spec, but uses a common [negative margin workaround](https://seek-oss.github.io/capsize/) under the hood for cross-browser support.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Heading     trim="normal"     style={{       background: "var(--gray-a2)",       borderTop: "1px dashed var(--gray-a7)",       borderBottom: "1px dashed var(--gray-a7)",     }}   &gt;     Without trim   &lt;/Heading&gt;    &lt;Heading     trim="both"     style={{       background: "var(--gray-a2)",       borderTop: "1px dashed var(--gray-a7)",       borderBottom: "1px dashed var(--gray-a7)",     }}   &gt;     With trim   &lt;/Heading&gt; &lt;/Flex&gt; ```  Trimming the leading is useful when dialing in vertical spacing in cards or other “boxy” components. Otherwise, padding looks larger on top and bottom than on the sides.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Box     style={{       background: "var(--gray-a2)",       border: "1px dashed var(--gray-a7)",     }}     p="4"   &gt;     &lt;Heading mb="1" size="3"&gt;       Without trim     &lt;/Heading&gt;      &lt;Text&gt;       The goal of typography is to relate font size, line height, and line width in a proportional       way that maximizes beauty and makes reading easier and more pleasant.     &lt;/Text&gt;   &lt;/Box&gt;    &lt;Box     p="4"     style={{       background: "var(--gray-a2)",       border: "1px dashed var(--gray-a7)",     }}   &gt;     &lt;Heading mb="1" size="3" trim="start"&gt;       With trim     &lt;/Heading&gt;      &lt;Text trim="end"&gt;       The goal of typography is to relate font size, line height, and line width in a proportional       way that maximizes beauty and makes reading easier and more pleasant.     &lt;/Text&gt;   &lt;/Box&gt; &lt;/Flex&gt; ```  The default trim values are configured for the system font stack that’s used by Radix Themes. If you are using custom fonts, you can [adjust](/themes/docs/theme/typography#customization) the trim values using the corresponding CSS variables.  ```css .radix-themes {   --default-leading-trim-start: 0.42em;    --default-leading-trim-end: 0.36em;    --heading-leading-trim-start: 0.42em;    --heading-leading-trim-end: 0.36em; } ```  ### [Truncate](#truncate)  Use the `truncate` prop to truncate text with an ellipsis when it overflows its container.  ```jsx &lt;Flex maxWidth="300px"&gt;   &lt;Heading truncate&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  ### [Wrap](#wrap)  Use the `wrap` prop to control text wrapping.  ```jsx &lt;Flex maxWidth="300px"&gt;   &lt;Heading wrap="nowrap"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  ```jsx &lt;Flex maxWidth="300px"&gt;   &lt;Heading wrap="balance"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  ```jsx &lt;Flex maxWidth="300px"&gt;   &lt;Heading wrap="pretty"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  `text-wrap: pretty` is an experimental value that is&lt;!-- --&gt; [not yet supported in all browsers](https://developer.mozilla.org/en-US/docs/Web/CSS/text-wrap#browser_compatibility). However, it can be treated as a progressive enhancement for browsers that do support it.  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color). The text colors are designed to achieve at least Lc 60 APCA contrast over common background colors.  ```jsx &lt;Flex direction="column"&gt;   &lt;Heading color="indigo"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading color="cyan"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading color="orange"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt;    &lt;Heading color="crimson"&gt;The quick brown fox jumps over the lazy dog&lt;/Heading&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column"&gt;   &lt;Heading color="gray"&gt;The quick brown fox jumps over the lazy dog.&lt;/Heading&gt;    &lt;Heading color="gray" highContrast&gt;     The quick brown fox jumps over the lazy dog.   &lt;/Heading&gt; &lt;/Flex&gt; ```
