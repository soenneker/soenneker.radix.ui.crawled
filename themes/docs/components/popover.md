This component is based on the `div` element.  | Prop        | Type                                   | Default          | | ----------- | -------------------------------------- | ---------------- | | `asChild`   | `boolean`                              | No default value | | `size`      | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;` | `"2"`            | | `width`     | `Responsive&lt;string&gt;`                   | No default value | | `minWidth`  | `Responsive&lt;string&gt;`                   | No default value | | `maxWidth`  | `Responsive&lt;string&gt;`                   | `"480px"`        | | `height`    | `Responsive&lt;string&gt;`                   | No default value | | `minHeight` | `Responsive&lt;string&gt;`                   | No default value | | `maxHeight` | `Responsive&lt;string&gt;`                   | No default value |  ### [Close](#close)  Wraps the control that will close the popover.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control size of the popover. It will affect the `padding` and `border-radius` of the Content.
