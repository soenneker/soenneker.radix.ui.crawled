		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Alert Dialog  Modal confirmation dialog that interrupts the user and expects a response.  ```jsx &lt;AlertDialog.Root&gt;   &lt;AlertDialog.Trigger&gt;     &lt;Button color="red"&gt;Revoke access&lt;/Button&gt;   &lt;/AlertDialog.Trigger&gt;    &lt;AlertDialog.Content maxWidth="450px"&gt;     &lt;AlertDialog.Title&gt;Revoke access&lt;/AlertDialog.Title&gt;      &lt;AlertDialog.Description size="2"&gt;       Are you sure? This application will no longer be accessible and any existing sessions will be       expired.     &lt;/AlertDialog.Description&gt;      &lt;Flex gap="3" mt="4" justify="end"&gt;       &lt;AlertDialog.Cancel&gt;         &lt;Button variant="soft" color="gray"&gt;           Cancel         &lt;/Button&gt;       &lt;/AlertDialog.Cancel&gt;        &lt;AlertDialog.Action&gt;         &lt;Button variant="solid" color="red"&gt;           Revoke access         &lt;/Button&gt;       &lt;/AlertDialog.Action&gt;     &lt;/Flex&gt;   &lt;/AlertDialog.Content&gt; &lt;/AlertDialog.Root&gt; ```  ## [API Reference](#api-reference)  This component inherits parts and props from the [Alert Dialog primitive](/primitives/docs/components/alert-dialog) and is visually identical to [Dialog](/themes/docs/components/dialog), though with differing semantics and behavior.  ### [Root](#root)  Contains all the parts of the dialog.  ### [Trigger](#trigger)  Wraps the control that will open the dialog.  ### [Content](#content)  Contains the content of the dialog. This component is based on the `div` element.  | Prop        | Type                                   | Default          | | ----------- | -------------------------------------- | ---------------- | | `asChild`   | `boolean`                              | No default value | | `align`     | `"start" \| "center"`                  | `"center"`       | | `size`      | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;` | `"3"`            | | `width`     | `Responsive&lt;string&gt;`                   | No default value | | `minWidth`  | `Responsive&lt;string&gt;`                   | No default value | | `maxWidth`  | `Responsive&lt;string&gt;`                   | `"600px"`        | | `height`    | `Responsive&lt;string&gt;`                   | No default value | | `minHeight` | `Responsive&lt;string&gt;`                   | No default value | | `maxHeight` | `Responsive&lt;string&gt;`                   | No default value |  ### [Title](#title)  An accessible title that is announced when the dialog is opened. This part is based on the [Heading](/themes/docs/components/heading) component with a pre-defined font size and leading trim on top.  ### [Description](#description)  An optional accessible description that is announced when the dialog is opened. This part is based on the [Text](/themes/docs/components/text) component with a pre-defined font size.  If you want to remove the description entirely, remove this part and pass `aria-describedby={undefined}` to `Content`.  ### [Action](#action)  Wraps the control that will close the dialog. This should be distinguished visually from the `Cancel` control.  ### [Cancel](#cancel)  Wraps the control that will close the dialog. This should be distinguished visually from the `Action` control.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control size of the dialog. It will affect the `padding` and `border-radius` of the Content. Use it in conjunction with the `width`, `minWidth` and `maxWidth` props to control the width of the dialog.  ```jsx &lt;Flex gap="4" align="center"&gt;   &lt;AlertDialog.Root&gt;     &lt;AlertDialog.Trigger&gt;       &lt;Button variant="soft"&gt;Size 1&lt;/Button&gt;     &lt;/AlertDialog.Trigger&gt;      &lt;AlertDialog.Content size="1" maxWidth="300px"&gt;       &lt;Text as="p" trim="both" size="1"&gt;         The quick brown fox jumps over the lazy dog.       &lt;/Text&gt;     &lt;/AlertDialog.Content&gt;   &lt;/AlertDialog.Root&gt;    &lt;AlertDialog.Root&gt;     &lt;AlertDialog.Trigger&gt;       &lt;Button variant="soft"&gt;Size 2&lt;/Button&gt;     &lt;/AlertDialog.Trigger&gt;      &lt;AlertDialog.Content size="2" maxWidth="400px"&gt;       &lt;Text as="p" trim="both" size="2"&gt;         The quick brown fox jumps over the lazy dog.       &lt;/Text&gt;     &lt;/AlertDialog.Content&gt;   &lt;/AlertDialog.Root&gt;    &lt;AlertDialog.Root&gt;     &lt;AlertDialog.Trigger&gt;       &lt;Button variant="soft"&gt;Size 3&lt;/Button&gt;     &lt;/AlertDialog.Trigger&gt;      &lt;AlertDialog.Content size="3" maxWidth="500px"&gt;       &lt;Text as="p" trim="both" size="3"&gt;         The quick brown fox jumps over the lazy dog.       &lt;/Text&gt;     &lt;/AlertDialog.Content&gt;   &lt;/AlertDialog.Root&gt;    &lt;AlertDialog.Root&gt;     &lt;AlertDialog.Trigger&gt;       &lt;Button variant="soft"&gt;Size 4&lt;/Button&gt;     &lt;/AlertDialog.Trigger&gt;      &lt;AlertDialog.Content size="4"&gt;       &lt;Text as="p" trim="both" size="4"&gt;
