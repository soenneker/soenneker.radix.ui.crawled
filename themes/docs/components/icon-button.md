		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Icon Button  Button designed specifically for usage with a single icon.  ```jsx &lt;IconButton&gt;   &lt;MagnifyingGlassIcon width="18" height="18" /&gt; &lt;/IconButton&gt; ```  ## [API Reference](#api-reference)  This component is based on the `button` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `asChild`      | `boolean`                                            | No default value | | `size`         | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;`               | `"2"`            | | `variant`      | `enum`                                               | `"solid"`        | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value | | `loading`      | `boolean`                                            | `false`          |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the button.  ```jsx &lt;Flex align="center" gap="3"&gt;   &lt;IconButton size="1" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="15" height="15" /&gt;   &lt;/IconButton&gt;    &lt;IconButton size="2" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton size="3" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="22" height="22" /&gt;   &lt;/IconButton&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the button.  ```jsx &lt;Flex gap="3"&gt;   &lt;IconButton variant="classic"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton variant="solid"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton variant="surface"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton variant="outline"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt; &lt;/Flex&gt; ```  #### Ghost  Use the `ghost` variant to display a button without chrome. Ghost buttons behave like text in layout, as they use a negative margin to optically align themselves against their siblings while maintaining the padding in active and hover states.  ```jsx &lt;IconButton variant="ghost"&gt;   &lt;MagnifyingGlassIcon width="18" height="18" /&gt; &lt;/IconButton&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="3"&gt;   &lt;IconButton color="crimson" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton color="indigo" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton color="grass" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton color="orange" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Flex gap="3"&gt;     &lt;IconButton color="gray" variant="classic"&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="solid"&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="soft"&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="surface"&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="outline"&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;   &lt;/Flex&gt;    &lt;Flex gap="3"&gt;     &lt;IconButton color="gray" variant="classic" highContrast&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="solid" highContrast&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="soft" highContrast&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="surface" highContrast&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;      &lt;IconButton color="gray" variant="outline" highContrast&gt;       &lt;MagnifyingGlassIcon width="18" height="18" /&gt;     &lt;/IconButton&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex gap="3"&gt;   &lt;IconButton radius="none" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton radius="large" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton radius="full" variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt; &lt;/Flex&gt; ```  ### [Loading](#loading)  Use the `loading` prop to display a loading spinner in place of button content. The button will be disabled while loading.  ```jsx &lt;Flex gap="3"&gt;   &lt;IconButton loading variant="classic"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton loading variant="solid"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton loading variant="soft"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton loading variant="surface"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt;    &lt;IconButton loading variant="outline"&gt;     &lt;MagnifyingGlassIcon width="18" height="18" /&gt;   &lt;/IconButton&gt; &lt;/Flex&gt; ```
