		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Dropdown Menu  Menu representing a set of actions, triggered by a button.  ```jsx &lt;DropdownMenu.Root&gt;   &lt;DropdownMenu.Trigger&gt;     &lt;Button variant="soft"&gt;       Options       &lt;DropdownMenu.TriggerIcon /&gt;     &lt;/Button&gt;   &lt;/DropdownMenu.Trigger&gt;    &lt;DropdownMenu.Content&gt;     &lt;DropdownMenu.Item shortcut="⌘ E"&gt;Edit&lt;/DropdownMenu.Item&gt;      &lt;DropdownMenu.Item shortcut="⌘ D"&gt;Duplicate&lt;/DropdownMenu.Item&gt;      &lt;DropdownMenu.Separator /&gt;      &lt;DropdownMenu.Item shortcut="⌘ N"&gt;Archive&lt;/DropdownMenu.Item&gt;      &lt;DropdownMenu.Sub&gt;       &lt;DropdownMenu.SubTrigger&gt;More&lt;/DropdownMenu.SubTrigger&gt;        &lt;DropdownMenu.SubContent&gt;         &lt;DropdownMenu.Item&gt;Move to project…&lt;/DropdownMenu.Item&gt;          &lt;DropdownMenu.Item&gt;Move to folder…&lt;/DropdownMenu.Item&gt;          &lt;DropdownMenu.Separator /&gt;          &lt;DropdownMenu.Item&gt;Advanced options…&lt;/DropdownMenu.Item&gt;       &lt;/DropdownMenu.SubContent&gt;     &lt;/DropdownMenu.Sub&gt;      &lt;DropdownMenu.Separator /&gt;      &lt;DropdownMenu.Item&gt;Share&lt;/DropdownMenu.Item&gt;      &lt;DropdownMenu.Item&gt;Add to favorites&lt;/DropdownMenu.Item&gt;      &lt;DropdownMenu.Separator /&gt;      &lt;DropdownMenu.Item shortcut="⌘ ⌫" color="red"&gt;       Delete     &lt;/DropdownMenu.Item&gt;   &lt;/DropdownMenu.Content&gt; &lt;/DropdownMenu.Root&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Dropdown Menu primitive](/primitives/docs/components/dropdown-menu).  ### [Root](#root)  Contains all the parts of a dropdown menu.  ### [Trigger](#trigger)  Wraps the control that will open the dropdown menu.  ### [Trigger Icon](#trigger-icon)  An optional icon part.  ### [Content](#content)  The component that pops out when the dropdown menu is open.  | Prop           | Type                     | Default          | | -------------- | ------------------------ | ---------------- | | `size`         | `Responsive&lt;"1" \| "2"&gt;` | `"2"`            | | `variant`      | `"solid" \| "soft"`      | `"solid"`        | | `color`        | `enum`                   | No default value | | `highContrast` | `boolean`                | No default value |  ### [Label](#label)  Used to render a label. It won't be focusable using arrow keys.  ### [Item](#item)  The component that contains the dropdown menu items.  | Prop       | Type      | Default          | | ---------- | --------- | ---------------- | | `asChild`  | `boolean` | No default value | | `color`    | `enum`    | No default value | | `shortcut` | `string`  | No default value |  ### [Group](#group)  Used to group multiple `Item` parts.  ### [RadioGroup](#radiogroup)  Used to group multiple `RadioItem` parts.  ### [RadioItem](#radioitem)  An item that can be controlled and rendered like a radio.  | Prop    | Type   | Default          | | ------- | ------ | ---------------- | | `color` | `enum` | No default value |  ### [CheckboxItem](#checkboxitem)  An item that can be controlled and rendered like a checkbox.  | Prop       | Type     | Default          | | ---------- | -------- | ---------------- | | `color`    | `enum`   | No default value | | `shortcut` | `string` | No default value |  ### [Sub](#sub)  Contains all the parts of a submenu.  ### [SubTrigger](#subtrigger)  An item that opens a submenu. Must be rendered inside `DropdownMenu.Sub`.  ### [SubContent](#subcontent)  The component that pops out when a submenu is open. Must be rendered inside `DropdownMenu.Sub`.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex gap="3" align="center"&gt;   &lt;DropdownMenu.Root&gt;     &lt;/DropdownMenu.Content&gt;   &lt;/DropdownMenu.Root&gt;    &lt;DropdownMenu.Root&gt;     &lt;DropdownMenu.Trigger&gt;       &lt;Button variant="soft" size="2"&gt;         Options         &lt;DropdownMenu.TriggerIcon /&gt;       &lt;/Button&gt;     &lt;/DropdownMenu.Trigger&gt;      &lt;DropdownMenu.Content size="2"&gt;       &lt;DropdownMenu.Item shortcut="⌘ E"&gt;Edit&lt;/DropdownMenu.Item&gt;        &lt;DropdownMenu.Item shortcut="⌘ D"&gt;Duplicate&lt;/DropdownMenu.Item&gt;         Delete       &lt;/DropdownMenu.Item&gt;     &lt;/DropdownMenu.Content&gt;   &lt;/DropdownMenu.Root&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to customize the visual style of the dropdown menu.  ```jsx &lt;Flex gap="3" align="center"&gt;
