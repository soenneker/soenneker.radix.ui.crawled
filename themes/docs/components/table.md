This component is based on the `table` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop      | Type                            | Default          | | --------- | ------------------------------- | ---------------- | | `size`    | `Responsive&lt;"1" \| "2" \| "3"&gt;` | `"2"`            | | `variant` | `"surface" \| "ghost"`          | `"ghost"`        | | `layout`  | `Responsive&lt;"auto" \| "fixed"&gt;` | No default value |  ### [Header](#header)  Contains the column headings for the table, based on the `thead` element.  ### [Body](#body)  Displays the table data. This component is based on the `tbody` element.  ### [Row](#row)  A row of table cells. Based on the `tr` element.  | Prop    | Type                                                     | Default          | | ------- | -------------------------------------------------------- | ---------------- | | `align` | `Responsive&lt;"start" \| "center" \| "end" \| "baseline"&gt;` | No default value |  ### [Cell](#cell)  A basic table cell. Based on the `th` element and provides the same props interface as the `Cell` part.  | Prop       | Type                                       | Default          | | ---------- | ------------------------------------------ | ---------------- | | `justify`  | `Responsive&lt;"start" \| "center" \| "end"&gt;` | No default value | | `width`    | `Responsive&lt;string&gt;`                       | No default value | | `minWidth` | `Responsive&lt;string&gt;`                       | No default value | | `maxWidth` | `Responsive&lt;string&gt;`                       | No default value | | `p`        | `Responsive&lt;enum \| string&gt;`               | No default value | | `px`       | `Responsive&lt;enum \| string&gt;`               | No default value | | `py`       | `Responsive&lt;enum \| string&gt;`               | No default value | | `pt`       | `Responsive&lt;enum \| string&gt;`               | No default value | | `pr`       | `Responsive&lt;enum \| string&gt;`               | No default value | | `pb`       | `Responsive&lt;enum \| string&gt;`               | No default value | | `pl`       | `Responsive&lt;enum \| string&gt;`               | No default value |  ### [RowHeaderCell](#rowheadercell)  The header of a table row.
