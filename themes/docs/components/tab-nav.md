		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Tab Nav  Navigation menu with links styled as tabs.  ```jsx &lt;TabNav.Root&gt;   &lt;TabNav.Link href="#" active&gt;     Account   &lt;/TabNav.Link&gt;    &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;    &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt; &lt;/TabNav.Root&gt; ```  ## [API Reference](#api-reference)  This component is based on the [Navigation Menu primitive](/primitives/docs/components/navigation-menu) and supports [common margin props](/themes/docs/overview/layout#margin-props).  ### [Root](#root)  Contains the navigation menu links.  | Prop           | Type                                               | Default          | | -------------- | -------------------------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2"&gt;`                           | `"2"`            | | `wrap`         | `Responsive&lt;"nowrap" \| "wrap" \| "wrap-reverse"&gt;` | No default value | | `justify`      | `Responsive&lt;"start" \| "center" \| "end"&gt;`         | No default value | | `color`        | `enum`                                             | No default value | | `highContrast` | `boolean`                                          | No default value |  ### [Link](#link)  An individual navigation menu link.  | Prop      | Type      | Default          | | --------- | --------- | ---------------- | | `asChild` | `boolean` | No default value | | `active`  | `boolean` | `false`          |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the tabs.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;TabNav.Root size="1"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt;    &lt;TabNav.Root size="2"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color) to the tab list.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;TabNav.Root color="indigo"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt;    &lt;TabNav.Root color="cyan"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt;    &lt;TabNav.Root color="orange"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt;    &lt;TabNav.Root color="crimson"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast with the background.  ```jsx &lt;Flex direction="column" gap="4" pb="2"&gt;   &lt;TabNav.Root color="gray"&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt;    &lt;TabNav.Root color="gray" highContrast&gt;     &lt;TabNav.Link href="#" active&gt;       Account     &lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Documents&lt;/TabNav.Link&gt;      &lt;TabNav.Link href="#"&gt;Settings&lt;/TabNav.Link&gt;   &lt;/TabNav.Root&gt; &lt;/Flex&gt; ```  ### [With router links](#with-router-links)  Use the `asChild` prop to compose `TabNav.Link` with your app’s router link component.  ```jsx &lt;TabNav.Root&gt;   &lt;TabNav.Link asChild active={pathname === "/account"}&gt;     &lt;NextLink href="/account"&gt;Account&lt;/NextLink&gt;   &lt;/TabNav.Link&gt;    &lt;TabNav.Link asChild active={pathname === "/documents"}&gt;     &lt;NextLink href="/documents"&gt;Documents&lt;/NextLink&gt;   &lt;/TabNav.Link&gt;    &lt;TabNav.Link asChild active={pathname === "/settings"}&gt;     &lt;NextLink href="/settings"&gt;Settings&lt;/NextLink&gt;   &lt;/TabNav.Link&gt; &lt;/TabNav.Root&gt; ```
