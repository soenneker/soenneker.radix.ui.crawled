Note that this dialog is designed around the modal pattern, so the&lt;!-- --&gt; `modal` prop is unavailable.  ### [Root](#root)  Contains all the parts of a dialog.  ### [Trigger](#trigger)  Wraps the control that will open the dialog.  ### [Content](#content)  Contains the content of the dialog. This component is based on the `div` element.  | Prop        | Type                                   | Default          | | ----------- | -------------------------------------- | ---------------- | | `asChild`   | `boolean`                              | No default value | | `align`     | `"start" \| "center"`                  | `"center"`       | | `size`      | `Responsive&lt;"1" \| "2" \| "3" \| "4"&gt;` | `"3"`            | | `width`     | `Responsive&lt;string&gt;`                   | No default value | | `minWidth`  | `Responsive&lt;string&gt;`                   | No default value | | `maxWidth`  | `Responsive&lt;string&gt;`                   | `"600px"`        | | `height`    | `Responsive&lt;string&gt;`                   | No default value | | `minHeight` | `Responsive&lt;string&gt;`                   | No default value | | `maxHeight` | `Responsive&lt;string&gt;`                   | No default value |  ### [Title](#title)  An accessible title that is announced when the dialog is opened. This part is based on the [Heading](/themes/docs/components/heading) component with a pre-defined font size and leading trim on top.  ### [Description](#description)  An optional accessible description that is announced when the dialog is opened. This part is based on the [Text](/themes/docs/components/text) component with a pre-defined font size.  If you want to remove the description entirely, remove this part and pass `aria-describedby={undefined}` to `Content`.  ### [Close](#close)  Wraps the control that will close the dialog.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control size of the dialog. It will affect the `padding` and `border-radius` of the Content. Use it in conjunction with the `width`, `minWidth` and `maxWidth` props to control the width of the dialog.  ```jsx &lt;Flex gap="4" align="center"&gt;   &lt;Dialog.Root&gt;     &lt;Dialog.Trigger&gt;
