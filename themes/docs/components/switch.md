		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Switch  Toggle switch alternative to the checkbox.  ```jsx &lt;Switch defaultChecked /&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Switch primitive](/primitives/docs/components/switch) and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"`                   | `"surface"`      | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size of the switch.  ```jsx &lt;Flex align="center" gap="2"&gt;   &lt;Switch size="1" defaultChecked /&gt;    &lt;Switch size="2" defaultChecked /&gt;    &lt;Switch size="3" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style of the switch.  ```jsx &lt;Flex gap="2"&gt;   &lt;Flex direction="column" gap="3"&gt;     &lt;Switch variant="surface" /&gt;      &lt;Switch variant="classic" /&gt;      &lt;Switch variant="soft" /&gt;   &lt;/Flex&gt;    &lt;Flex direction="column" gap="3"&gt;     &lt;Switch variant="surface" defaultChecked /&gt;      &lt;Switch variant="classic" defaultChecked /&gt;      &lt;Switch variant="soft" defaultChecked /&gt;   &lt;/Flex&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex gap="2"&gt;   &lt;Switch color="indigo" defaultChecked /&gt;    &lt;Switch color="cyan" defaultChecked /&gt;    &lt;Switch color="orange" defaultChecked /&gt;    &lt;Switch color="crimson" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast in light mode.  ```jsx &lt;Grid rows="2" gapX="2" gapY="3" display="inline-grid" flow="column"&gt;   &lt;Switch color="indigo" defaultChecked /&gt;    &lt;Switch color="indigo" defaultChecked highContrast /&gt;    &lt;Switch color="cyan" defaultChecked /&gt;    &lt;Switch color="cyan" defaultChecked highContrast /&gt;    &lt;Switch color="orange" defaultChecked /&gt;    &lt;Switch color="orange" defaultChecked highContrast /&gt;    &lt;Switch color="crimson" defaultChecked /&gt;    &lt;Switch color="crimson" defaultChecked highContrast /&gt;    &lt;Switch color="gray" defaultChecked /&gt;    &lt;Switch color="gray" defaultChecked highContrast /&gt; &lt;/Grid&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex gap="3"&gt;   &lt;Switch radius="none" defaultChecked /&gt;    &lt;Switch radius="small" defaultChecked /&gt;    &lt;Switch radius="full" defaultChecked /&gt; &lt;/Flex&gt; ```  ### [Alignment](#alignment)  Composing `Switch` within `Text` automatically centers it with the first line of text.  ```jsx &lt;Flex direction="column" gap="3"&gt;   &lt;Text as="label" size="2"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="1" defaultChecked /&gt; Sync settings     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="3"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="2" defaultChecked /&gt; Sync settings     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="4"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="3" defaultChecked /&gt; Sync settings     &lt;/Flex&gt;   &lt;/Text&gt; &lt;/Flex&gt; ```  It is automatically well-aligned with multi-line text too.  ### [Disabled](#disabled)  Use the native `disabled` attribute to create a disabled switch.  ```jsx &lt;Flex direction="column" gap="2"&gt;   &lt;Text as="label" size="2"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="1" /&gt;       Off     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="1" defaultChecked /&gt;       On     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2" color="gray"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="1" disabled /&gt;       On     &lt;/Flex&gt;   &lt;/Text&gt;    &lt;Text as="label" size="2" color="gray"&gt;     &lt;Flex gap="2"&gt;       &lt;Switch size="1" disabled defaultChecked /&gt;       Off     &lt;/Flex&gt;   &lt;/Text&gt; &lt;/Flex&gt; ```
