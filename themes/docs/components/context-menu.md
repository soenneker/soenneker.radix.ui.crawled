		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Context Menu  Menu representing a set of actions, displayed at the point of right click or long press.  ```jsx &lt;ContextMenu.Root&gt;   &lt;ContextMenu.Trigger&gt;     &lt;RightClickZone style={{ height: 150 }} /&gt;   &lt;/ContextMenu.Trigger&gt;    &lt;ContextMenu.Content&gt;     &lt;ContextMenu.Item shortcut="⌘ E"&gt;Edit&lt;/ContextMenu.Item&gt;      &lt;ContextMenu.Item shortcut="⌘ D"&gt;Duplicate&lt;/ContextMenu.Item&gt;      &lt;ContextMenu.Separator /&gt;      &lt;ContextMenu.Item shortcut="⌘ N"&gt;Archive&lt;/ContextMenu.Item&gt;      &lt;ContextMenu.Sub&gt;       &lt;ContextMenu.SubTrigger&gt;More&lt;/ContextMenu.SubTrigger&gt;        &lt;ContextMenu.SubContent&gt;         &lt;ContextMenu.Item&gt;Move to project…&lt;/ContextMenu.Item&gt;          &lt;ContextMenu.Item&gt;Move to folder…&lt;/ContextMenu.Item&gt;          &lt;ContextMenu.Separator /&gt;          &lt;ContextMenu.Item&gt;Advanced options…&lt;/ContextMenu.Item&gt;       &lt;/ContextMenu.SubContent&gt;     &lt;/ContextMenu.Sub&gt;      &lt;ContextMenu.Separator /&gt;      &lt;ContextMenu.Item&gt;Share&lt;/ContextMenu.Item&gt;      &lt;ContextMenu.Item&gt;Add to favorites&lt;/ContextMenu.Item&gt;      &lt;ContextMenu.Separator /&gt;      &lt;ContextMenu.Item shortcut="⌘ ⌫" color="red"&gt;       Delete     &lt;/ContextMenu.Item&gt;   &lt;/ContextMenu.Content&gt; &lt;/ContextMenu.Root&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Context Menu primitive](/primitives/docs/components/context-menu).  ### [Root](#root)  Contains all the parts of a context menu.  ### [Trigger](#trigger)  Wraps the element that will open the context menu.  ### [Content](#content)  The component that pops out when the context menu is open.  | Prop           | Type                     | Default          | | -------------- | ------------------------ | ---------------- | | `size`         | `Responsive&lt;"1" \| "2"&gt;` | `"2"`            | | `variant`      | `"solid" \| "soft"`      | `"solid"`        | | `color`        | `enum`                   | No default value | | `highContrast` | `boolean`                | No default value |  ### [Label](#label)  Used to render a label. It won't be focusable using arrow keys.  ### [Item](#item)  The component that contains the context menu items.  | Prop       | Type      | Default          | | ---------- | --------- | ---------------- | | `asChild`  | `boolean` | No default value | | `color`    | `enum`    | No default value | | `shortcut` | `string`  | No default value |  ### [Group](#group)  Used to group multiple `Item` parts.  ### [RadioGroup](#radiogroup)  Used to group multiple `RadioItem` parts.  ### [RadioItem](#radioitem)  An item that can be controlled and rendered like a radio.  | Prop    | Type   | Default          | | ------- | ------ | ---------------- | | `color` | `enum` | No default value |  ### [CheckboxItem](#checkboxitem)  An item that can be controlled and rendered like a checkbox.  | Prop       | Type     | Default          | | ---------- | -------- | ---------------- | | `color`    | `enum`   | No default value | | `shortcut` | `string` | No default value |  ### [Sub](#sub)  Contains all the parts of a submenu.  ### [SubTrigger](#subtrigger)  An item that opens a submenu. Must be rendered inside `ContextMenu.Sub`.  ### [SubContent](#subcontent)  The component that pops out when a submenu is open. Must be rendered inside `ContextMenu.Sub`.  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Grid columns="2" gap="3"&gt;   &lt;ContextMenu.Root&gt;     &lt;ContextMenu.Trigger&gt;       &lt;RightClickZone title="Size one" /&gt;     &lt;/ContextMenu.Trigger&gt;      &lt;ContextMenu.Content size="1"&gt;       &lt;ContextMenu.Item shortcut="⌘ E"&gt;Edit&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Item shortcut="⌘ D"&gt;Duplicate&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Separator /&gt;        &lt;ContextMenu.Item shortcut="⌘ N"&gt;Archive&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Separator /&gt;        &lt;ContextMenu.Item shortcut="⌘ ⌫" color="red"&gt;         Delete       &lt;/ContextMenu.Item&gt;     &lt;/ContextMenu.Content&gt;   &lt;/ContextMenu.Root&gt;    &lt;ContextMenu.Root&gt;     &lt;ContextMenu.Trigger&gt;       &lt;RightClickZone title="Size two" /&gt;     &lt;/ContextMenu.Trigger&gt;      &lt;ContextMenu.Content size="2"&gt;       &lt;ContextMenu.Item shortcut="⌘ E"&gt;Edit&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Item shortcut="⌘ D"&gt;Duplicate&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Separator /&gt;        &lt;ContextMenu.Item shortcut="⌘ N"&gt;Archive&lt;/ContextMenu.Item&gt;        &lt;ContextMenu.Separator /&gt;        &lt;ContextMenu.Item shortcut="⌘ ⌫" color="red"&gt;         Delete       &lt;/ContextMenu.Item&gt;     &lt;/ContextMenu.Content&gt;   &lt;/ContextMenu.Root&gt; &lt;/Grid&gt; ```  ### [Variant](#variant)  Use the `variant` prop to customize the visual style of the context menu.  ```jsx &lt;Grid columns="2" gap="3"&gt;
