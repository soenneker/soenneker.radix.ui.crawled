		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Skeleton  Replaces content with same shape placeholder that indicates a loading state.  ```jsx &lt;Skeleton&gt;Loading&lt;/Skeleton&gt; ```  ## [API Reference](#api-reference)  This component is based on the `span` element and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop        | Type                 | Default          | | ----------- | -------------------- | ---------------- | | `loading`   | `boolean`            | `true`           | | `width`     | `Responsive&lt;string&gt;` | No default value | | `minWidth`  | `Responsive&lt;string&gt;` | No default value | | `maxWidth`  | `Responsive&lt;string&gt;` | No default value | | `height`    | `Responsive&lt;string&gt;` | No default value | | `minHeight` | `Responsive&lt;string&gt;` | No default value | | `maxHeight` | `Responsive&lt;string&gt;` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the width and height props to manually control the size of the skeleton.  ```jsx &lt;Skeleton width="48px" height="48px" /&gt; ```  ### [With children](#with-children)  Use the `loading` prop to control whether the skeleton or its children are displayed. Skeleton preserves the dimensions of children when they are hidden and disables interactive elements.  ```jsx &lt;Flex gap="4"&gt;   &lt;Skeleton loading={true}&gt;     &lt;Switch defaultChecked /&gt;   &lt;/Skeleton&gt;    &lt;Skeleton loading={false}&gt;     &lt;Switch defaultChecked /&gt;   &lt;/Skeleton&gt; &lt;/Flex&gt; ```  ### [With text](#with-text)  When using Skeleton with text, you’d usually wrap the text node itself rather than the parent element. This ensures that the text is replaced with a placeholder of the same size:  ```jsx &lt;Container size="1"&gt;   &lt;Flex direction="column" gap="2"&gt;     &lt;Text&gt;       &lt;Skeleton&gt;Lorem ipsum dolor sit amet.&lt;/Skeleton&gt;     &lt;/Text&gt;      &lt;Skeleton&gt;       &lt;Text&gt;Lorem ipsum dolor sit amet&lt;/Text&gt;     &lt;/Skeleton&gt;   &lt;/Flex&gt; &lt;/Container&gt; ```  The difference is especially noticeable when wrapping longer paragraphs:  ```jsx &lt;Container size="1"&gt;   &lt;Flex direction="column" gap="3"&gt;     &lt;Text&gt;       &lt;Skeleton&gt;         Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nam magna erat, fringilla sed commodo sed,         aliquet nec magna.       &lt;/Skeleton&gt;     &lt;/Text&gt;      &lt;Skeleton&gt;       &lt;Text&gt;         Lorem ipsum dolor sit amet, consectetur adipiscing elit.
