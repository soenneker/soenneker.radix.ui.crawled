		<pre style="word-wrap: break-word; white-space: pre-wrap;"># Slider  Provides user selection from a range of values.  ```jsx &lt;Slider defaultValue={[50]} /&gt; ```  ## [API Reference](#api-reference)  This component inherits props from the [Slider primitive](/primitives/docs/components/slider) and supports [common margin props](/themes/docs/overview/layout#margin-props).  | Prop           | Type                                                 | Default          | | -------------- | ---------------------------------------------------- | ---------------- | | `size`         | `Responsive&lt;"1" \| "2" \| "3"&gt;`                      | `"2"`            | | `variant`      | `"classic" \| "surface" \| "soft"`                   | `"surface"`      | | `color`        | `enum`                                               | No default value | | `highContrast` | `boolean`                                            | No default value | | `radius`       | `"none" \| "small" \| "medium" \| "large" \| "full"` | No default value |  ## [Examples](#examples)  ### [Size](#size)  Use the `size` prop to control the size.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Slider defaultValue={[25]} size="1" /&gt;    &lt;Slider defaultValue={[50]} size="2" /&gt;    &lt;Slider defaultValue={[75]} size="3" /&gt; &lt;/Flex&gt; ```  ### [Variant](#variant)  Use the `variant` prop to control the visual style.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Slider defaultValue={[25]} variant="surface" /&gt;    &lt;Slider defaultValue={[50]} variant="classic" /&gt;    &lt;Slider defaultValue={[75]} variant="soft" /&gt; &lt;/Flex&gt; ```  ### [Color](#color)  Use the `color` prop to assign a specific [color](/themes/docs/theme/color).  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Slider defaultValue={[20]} color="indigo" /&gt;    &lt;Slider defaultValue={[40]} color="cyan" /&gt;    &lt;Slider defaultValue={[60]} color="orange" /&gt;    &lt;Slider defaultValue={[80]} color="crimson" /&gt; &lt;/Flex&gt; ```  ### [High-contrast](#high-contrast)  Use the `highContrast` prop to increase color contrast in light mode.  ```jsx &lt;Grid columns="2" gap="4"&gt;   &lt;Slider defaultValue={[10]} color="indigo" /&gt;    &lt;Slider defaultValue={[10]} color="indigo" highContrast /&gt;    &lt;Slider defaultValue={[30]} color="cyan" /&gt;    &lt;Slider defaultValue={[30]} color="cyan" highContrast /&gt;    &lt;Slider defaultValue={[50]} color="orange" /&gt;    &lt;Slider defaultValue={[50]} color="orange" highContrast /&gt;    &lt;Slider defaultValue={[70]} color="crimson" /&gt;    &lt;Slider defaultValue={[70]} color="crimson" highContrast /&gt;    &lt;Slider defaultValue={[90]} color="gray" /&gt;    &lt;Slider defaultValue={[90]} color="gray" highContrast /&gt; &lt;/Grid&gt; ```  ### [Radius](#radius)  Use the `radius` prop to assign a specific radius value.  ```jsx &lt;Flex direction="column" gap="4" maxWidth="300px"&gt;   &lt;Slider defaultValue={[25]} radius="none" /&gt;    &lt;Slider defaultValue={[50]} radius="small" /&gt;    &lt;Slider defaultValue={[75]} radius="full" /&gt; &lt;/Flex&gt; ```  ### [Range](#range)  Provide multiple values to create a range slider.  ```jsx &lt;Slider defaultValue={[25, 75]} /&gt; ```
